# Suggested Commands

## Build & Test
- `swift build` (debug)
- `swift build -c release`
- `swift test`

## Run CLI
- `swift run gabgab-cli tts "Hello" --output greeting.wav`
- `swift run gabgab-cli stt recording.wav`
- `swift run gabgab-cli health`

## Run MCP Server
- `swift run gabgab-mcp`

## Utilities (macOS)
- `git status`
- `ls`
- `pwd`
- `grep` (prefer tool-based search when using agents)
